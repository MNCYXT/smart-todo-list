# 智能待办清单应用

一个基于Vue3的智能待办清单管理应用，具有任务管理、分类筛选、优先级标记和数据统计等功能。

## 功能特性

- ✅ 任务增删改查
- ✅ 分类标签管理
- ✅ 优先级标记（高/中/低）
- ✅ 本地存储（localStorage）
- ✅ 数据统计图表
- ✅ 响应式设计

## 技术栈

- **前端框架**: Vue 3
- **UI组件库**: Element Plus
- **图表库**: ECharts
- **构建工具**: Vite
- **存储**: localStorage

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```
